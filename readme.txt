Домашнее задание

Иммутабельные структуры данных: Option и List

Цель:

На практике освоить некоторые приемы работы с такими иммутабельными структурами данных как Option и List.


Описание/Пошаговая инструкция выполнения домашнего задания:

Во время вебинара мы вместе с вами разработали свою упрощенную версию структуры данных Option. Для нее мы уже реализовали несколько методов.
Ваша задача реализовать оставшиеся нереализованные методы используя подходы рассмотренные на занятиях (рекурсия, паттерн матчинг):
Option
-printIfAny
-zip
-filter
Также вам необходимо реализовать односвязный иммутабельный список
Как и Option список (List)  может быть представлен двумя случаями:
Nil - пустой список
Cons - не пустой список, содержит первый элемент(голову) и хвост (оставшийся список)
Для этой структуры данных реализуйте набор методов:

• ::

• apply

• mkString

• reverse

• map

• flatMap

• filter

• incList

• shoutString
Для этого необходимо:

• скачать репозиторий https://github.com/ScalaOtus/scala-dev-mooc-2023-12.git

• выполнить выше описанные задачи в файле (module1/03-introducing_fp_concepts.scala)
В файле есть все описания для выше указанных методов.
Старайтесь использовать приемы рассмотренные на занятии (рекурсия, паттерн матчинг)
Задание оправлять ссылкой на мерж реквест на склонированный к вам репозиторий.
