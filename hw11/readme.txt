Домашнее задание

Реализация CRUD в DAO

Цель:

Выполнив домашнее задание Вы попрактикуетесь в реализации DAO слоя, а также с его помощью реализуете сервис для бизнес слоя. В ДЗ используются ZIO, Quill, Testcontainers
#zio #zio services #quill #testcontainers


Описание/Пошаговая инструкция выполнения домашнего задания:

Для запуска тестов из ДЗ понадобиться скачать и установить docker-compose https://docs.docker.com/compose/install/
Скачайте или обновите проект https://github.com/ScalaOtus/scala-dev-mooc-2023-12
В папке test/module4 вы найдете весь инфраструктурный код. Также там находятся 2 файла с тестами UserRepositorySpec и UserServiceSpec
Для успешной сдачи ДЗ все тесты должны успешно запускаться и проходить.
Тесты можно запустить из вашего IDE или же из консоли с помощью команды sbt test.
Обращаю ваше внимание, что первый запуск может занять больше времени, т.к. будут качаться докер образы.
Для того, чтобы тесты запустились и прошли, вам нужно реализовать:

• UserRepository

• UserService
Они представляют собой классические ZIO сервисы с готовыми интерфейсами
Реализовывая метод сервиса addUserWithRole, подумайте о транзакционности.


Критерии оценки:

После реализации тестов src/test/scala/ru/otus/jdbc/homework/UserDaoTest.scala должны выполняться.
"Принято" - задание выполнено полностью
"Возвращено на доработку" - задание не выполнено полностью
